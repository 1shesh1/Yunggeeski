# Client / partner brand assets

Logos shown in the `/brands` case study section.

Expected files (referenced from `lib/sponsorship.ts`):

- `delta-options.png` — Delta Options logo

Notes:

- Logos render on a **white chip**, so a transparent PNG with dark artwork is
  ideal; a logo with its own light background also works.
- Aim for ~2x the display size (rendered at ~56px tall), e.g. **~480×120**.
- If a file is missing, the case study falls back to the client name as text —
  the page never breaks.
