# Downloadable files

Any file you put in this folder is served at:

**https://yoursite.com/downloads/yourfile.pdf**

Examples:
- `public/downloads/guide.pdf` → `/downloads/guide.pdf`
- `public/downloads/terms.pdf` → `/downloads/terms.pdf`

Add your files here, then add links on your site (e.g. the Downloads page or in the footer).
